## Метрики успеха (Gold Project Targets)

Эффективность Moonshot-целей измеряется на эталонном проекте по следующим KPI:

1. **Architectural Clarity:** 10/10 разработчиков понимают коренную причину (Root Cause) проблемы без обращения к документации.
2. **Exploit Reliability:** 90% подтвержденных находок уровня `BLOCKER` имеют воспроизводимый PoC в изолированной среде.
3. **Semantic Recall:** Нахождение 0-day уязвимостей в логике, которые не покрываются стандартными паттернами CVE.
4. **Surface Coverage:** 100% обнаружение всех активных роутов, включая те, что отсутствуют в документации.
5. **Zero-Touch Compliance:** Генерация готового Audit-ready отчета, требующего менее 5% правок человеком.
